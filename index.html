Bem-vindo ao SisAgro

Gestão Integrada para Agropecuária

SisAgro 1.0 - Gestão para o Campo

Sistema desenvolvido para integrar aprendizado técnico e gestão de agropecuária, conectando Web e Desktop ao banco SQLite. Ideal para fazendas, revendas de insumos e cooperativas.

🚀 Módulos do Sistema:

Ícone Descrição
👨‍🌾 Produtores Rurais: Cadastro de clientes e histórico de compras.
🌾 Insumos Agrícolas: Controle de sementes, rações, fertilizantes e equipamentos.
💰 Vendas: Registro de vendas de insumos e orçamentos para o campo.

📊 Tecnologias

  • Backend: Node.js com Express.
  • Banco: SQLite (sisagro.db).
  • Desktop: Visual Basic .NET (Módulo de Conexão).

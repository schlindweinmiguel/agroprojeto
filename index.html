Bem-vindo ao AgroVerde

Gestão Integrada para Agropecuária

AgroVerde 1.0 • Unidade Capanema

Sistema desenvolvido para integrar aprendizado técnico e a gestão de fazendas e comércios agropecuários, conectando Web e Desktop ao banco SQLite.

→ Módulos do Sistema

  • Clientes: Cadastro e histórico de produtores rurais e clientes.
  • Produtos: Controle de estoque de rações, sementes, fertilizantes e equipamentos.
  • Vendas: Registro de transações, orçamentos e baixa automática de estoque.

→ Tecnologias

  • Backend: Node.js com Express
  • Banco de Dados: SQLite (sisagro.db)
  • Desktop: Visual Basic .NET
